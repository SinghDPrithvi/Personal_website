High-Level Introduction to Reinforcement Learning

Reinforcement Learning is subset of machine learning that focuses on learning through rewards and experience. Unlike other subsets to machine learning like Supervised learning(eg. Deep Learning) their is a higher possibility for error and mistakes that could lead to the agent behaving in a way totally oppsite of how we wanted it to behave. Though the upside to it is that it dosen't need past data to train the agent which the other subset of machine learning do, all that RL needs is a environment to prompt it with rewards, observation for it to optmize its actions. Although this is a very simple explanation I wanted to get you excited about RL because the way it learns is very similar to how we as humans learn and it holds a lot of potential. RL has already shown promise in robotics, gaming and used to tune LLM like chatgpt and claude such that they don't prompt out BS and obsecne content.
